This program lets user generate passwords from four password types: strong, medium, weak and "I want 2 b hacked". These correspond to 32 chars (letters + digits + symbols), 16 chars (letters + digits + symbols), 16 chars (letters + digits) and 8 chars (letters), respectively. Simply select one of the options from the menu to generate a password.
